01 · ¿Qué es CSS3?

CSS significa Cascading Style Sheets (Hojas de Estilo en Cascada). Es el lenguaje que controla cómo se ve todo lo que está estructurado en HTML: colores, tamaños, tipografías, espaciados, animaciones y layouts.

🎨

Analogía: Si HTML es la estructura de una casa (paredes, puertas, ventanas), CSS es todo lo demás: la pintura, los muebles, la iluminación, las cortinas y el jardín. Sin CSS, la web sería solo texto negro sobre fondo blanco.

CSS3 es la versión más reciente, que se desarrolla en módulos independientes. Trajo Flexbox, Grid, variables nativas, animaciones, gradientes, fuentes web, filtros y media queries sin necesitar JavaScript ni herramientas externas.

Creado en 1996 por Håkon Wium Lie Estándar W3C Extensión .css Separación de contenido y presentación Funciona en todos los navegadores
3 formas de incluir CSS
/* 1. EXTERNO — el más recomendado */ /* En el <head> del HTML: */ <link rel="stylesheet" href="estilos.css"/> /* 2. INTERNO — dentro del <head> */ <style> body { background: #000; } </style> /* 3. EN LÍNEA — solo para casos muy específicos */ <p style="color: red; font-size: 18px;">Texto</p>

03 · Selectores

Los selectores determinan a qué elementos HTML se aplican los estilos. Son la parte más poderosa de CSS.

ElementoSelector de etiqueta

Aplica estilos a todas las instancias de esa etiqueta HTML en la página.

p { color: white; } h1 { font-size: 2rem; } a { text-decoration: none; }
.claseSelector de clase

Aplica estilos a todos los elementos con esa clase. El más usado. Se puede repetir en varios elementos.

.boton { background: #264de4; border-radius: 8px; }
#idSelector de ID

Aplica estilos a un elemento único con ese ID. Solo debe haber un elemento con ese ID por página.

#encabezado { height: 80px; position: sticky; }
A BSelector descendiente

Selecciona elementos B que estén dentro de A, sin importar qué tan anidados estén.

nav a { color: white; font-weight: 500; }
:hover :focusPseudoclases

Seleccionan elementos en un estado específico: cuando el cursor pasa, está enfocado, es el primero, etc.

button:hover { background: #0044bb; } input:focus { border-color: #264de4; } li:first-child { font-weight: bold; }
::before ::afterPseudoelementos

Crean elementos virtuales antes o después del contenido, sin modificar el HTML.

h2::before { content: "✦ "; color: #f7c5d5; } p::first-line { font-weight: bold; }

04 · El Modelo de Caja (Box Model)

Todo elemento HTML es una caja rectangular con cuatro capas. Entender el Box Model es fundamental para controlar el espaciado y tamaño de los elementos.

CSS — Box Model completo
.caja { /* Tamaño del contenido */ width: 300px; height: 150px; /* Relleno interno */ padding: 20px; /* todos los lados */ padding: 10px 20px; /* vertical horizontal */ padding-top: 10px; /* solo arriba */ /* Borde */ border: 2px solid #264de4; border-radius: 12px; /* esquinas redondeadas */ /* Margen externo */ margin: 0 auto; /* centrar horizontalmente */ margin-bottom: 2rem; /* IMPORTANTE: incluye padding y border en el width */ box-sizing: border-box; }

06 · Tipografía

CSS controla completamente cómo se ve el texto: familia tipográfica, tamaño, peso, interlineado, alineación y más.

CSS — Propiedades tipográficas
body { /* Familia de fuente con fallback */ font-family: 'DM Sans', Arial, sans-serif; font-size: 16px; /* tamaño base */ font-weight: 400; /* 100–900 o bold/normal */ font-style: normal; /* normal | italic | oblique */ line-height: 1.6; /* interlineado (sin unidad) */ letter-spacing: 0.05em; /* espaciado entre letras */ text-align: left; /* left|center|right|justify */ text-transform: none; /* uppercase|lowercase|capitalize */ text-decoration: none; /* underline|overline|line-through */ } /* Fuente de Google Fonts — en el HTML */ /* <link href="https://fonts.googleapis.com/css2?family= Playfair+Display&display=swap" rel="stylesheet"/> */ h1 { font-family: 'Playfair Display', serif; font-size: clamp(2rem, 5vw, 5rem); /* responsive font */ }

07 · Display y posicionamiento
displayTipo de visualización

Controla cómo se muestra un elemento en el flujo del documento.

.bloque { display: block; } .linea { display: inline; } .mixto { display: inline-block; } .oculto { display: none; } .flex { display: flex; } .cuadricula{ display: grid; }
positionPosicionamiento

Controla cómo se ubica el elemento respecto a su contenedor o al viewport.

.estatico { position: static; } .relativo { position: relative; } .absoluto { position: absolute; } .fijo { position: fixed; } .pegajoso { position: sticky; } .ejemplo { position: absolute; top: 20px; right: 0; }
overflow z-indexControl adicional

overflow controla qué pasa con el contenido que desborda. z-index controla la capa visual.

.contenedor { overflow: hidden; /* ocultar */ overflow: scroll; /* scroll */ overflow-x: auto; /* solo horizontal */ } .encima { position: relative; z-index: 10; }
08 · Flexbox

Flexbox es el sistema de layout unidimensional de CSS. Organiza elementos en una fila o columna con control total sobre alineación, espaciado y orden. Es el más usado para componentes y barras de navegación.

CSS — Flexbox
.contenedor { display: flex; /* Dirección */ flex-direction: row; /* row | column | row-reverse */ /* Alineación horizontal */ justify-content: space-between; /* flex-start|center|space-around */ /* Alineación vertical */ align-items: center; /* stretch|flex-start|flex-end */ /* Salto de línea si no caben */ flex-wrap: wrap; /* Espacio entre hijos */ gap: 1rem; } .hijo { flex: 1; /* crecer para llenar espacio */ flex-shrink: 0; /* no encoger */ flex-basis: 200px; /* tamaño base */ order: 2; /* reordenar visualmente */ }

09 · CSS Grid

CSS Grid es el sistema de layout bidimensional. A diferencia de Flexbox, controla filas y columnas al mismo tiempo. Ideal para estructuras de página completas y galerías.

CSS — Grid
.cuadricula { display: grid; /* Definir columnas */ grid-template-columns: 1fr 2fr 1fr; grid-template-columns: repeat(3, 1fr); grid-template-columns: repeat(auto-fill, minmax(250px, 1fr)); /* Filas */ grid-template-rows: auto 1fr auto; /* Espacio entre celdas */ gap: 1.5rem; } .elemento-ancho { grid-column: span 2; /* ocupa 2 columnas */ grid-row: span 2; /* ocupa 2 filas */ }

10 · Transiciones

Las transiciones animan el cambio de una propiedad CSS de un valor a otro, creando efectos suaves al pasar el cursor o cambiar de estado.

CSS — Transiciones
.boton { background: #264de4; color: white; padding: 0.6rem 1.4rem; border-radius: 8px; /* transition: propiedad duración función retardo */ transition: background 0.3s ease; /* Múltiples propiedades */ transition: background 0.3s ease, transform 0.2s ease, box-shadow 0.3s ease; /* O todas las propiedades */ transition: all 0.3s ease; } .boton:hover { background: #0033aa; transform: translateY(-4px); box-shadow: 0 12px 24px rgba(38,77,228,0.4); }

11 · Animaciones con @keyframes

Las animaciones CSS permiten crear efectos complejos que se reproducen automáticamente, sin necesitar JavaScript. Se definen con @keyframes y se aplican con animation.

CSS — @keyframes y animation
/* 1. Definir la animación */ @keyframes pulsar { 0% { transform: scale(1); opacity: 1; } 50% { transform: scale(1.2); opacity: 0.7; } 100% { transform: scale(1); opacity: 1; } } @keyframes deslizar { from { transform: translateX(-100%); opacity: 0; } to { transform: translateX(0); opacity: 1; } } /* 2. Aplicar la animación */ .elemento { /* animation: nombre duración función retardo iteraciones dirección */ animation: pulsar 2s ease-in-out 0s infinite; /* Propiedades individuales */ animation-name: deslizar; animation-duration: 0.5s; animation-fill-mode: forwards; /* mantiene el estado final */ animation-play-state: paused; /* pausar con JS */ }

12 · Diseño responsivo y Media Queries

El diseño responsivo adapta la apariencia de la página a cualquier tamaño de pantalla: celular, tablet o escritorio. Se logra con Media Queries y unidades relativas.

CSS — Media Queries y unidades relativas
/* ── Unidades relativas ── */ .elemento { width: 90%; /* % del contenedor padre */ font-size: 1rem; /* relativo al root (16px por defecto) */ padding: 2em; /* relativo al font-size del elemento */ height: 100vh; /* 100% del alto del viewport */ width: 50vw; /* 50% del ancho del viewport */ } /* ── Media Queries — Mobile First ── */ /* Estilos base: móvil */ .tarjetas { display: grid; grid-template-columns: 1fr; /* 1 columna en móvil */ gap: 1rem; } /* Tablet (768px o más) */ @media (min-width: 768px) { .tarjetas { grid-template-columns: repeat(2, 1fr); } } /* Escritorio (1024px o más) */ @media (min-width: 1024px) { .tarjetas { grid-template-columns: repeat(3, 1fr); } } /* Modo oscuro del sistema */ @media (prefers-color-scheme: dark) { body { background: #080808; } } /* Orientación horizontal */ @media (orientation: landscape) { .hero { min-height: 80vh; } }
13 · Variables CSS (Custom Properties)

Las variables CSS permiten almacenar valores reutilizables y crear sistemas de diseño coherentes. Se definen con --nombre y se usan con var(--nombre).

CSS — Variables y sistema de diseño
/* Definir variables globales en :root */ :root { /* Colores */ --color-primario: #264de4; --color-acento: #f7c5d5; --color-fondo: #080808; --color-texto: #f5f5f5; /* Tipografía */ --fuente-titulo: 'Playfair Display', serif; --fuente-cuerpo: 'DM Sans', sans-serif; --tam-base: 16px; /* Espaciado */ --espacio-sm: 0.5rem; --espacio-md: 1rem; --espacio-lg: 2rem; /* Bordes */ --radio-sm: 6px; --radio-lg: 14px; --radio-pill: 999px; } /* Usar las variables */ .boton { background: var(--color-primario); color: var(--color-texto); font-family: var(--fuente-cuerpo); padding: var(--espacio-sm) var(--espacio-md); border-radius: var(--radio-pill); } /* Modificar variables para modo oscuro */ [data-tema="claro"] { --color-fondo: #ffffff; --color-texto: #111111; }
14 · Buenas prácticas
✓ Hacer Mobile First

Diseña primero para móviles y usa min-width en tus media queries para escalar hacia pantallas más grandes.

/* ✓ Mobile first */ .nav { display: block; } @media (min-width: 768px) { .nav { display: flex; } }
✓ Hacer box-sizing: border-box

Aplica siempre esto globalmente para que el padding y border no rompan el tamaño definido.

*, *::before, *::after { box-sizing: border-box; }
✓ Hacer Usa variables CSS

Define colores, tipografías y espaciados como variables en :root para mantener consistencia y fácil mantenimiento.

:root { --azul: #264de4; } .btn { background: var(--azul); }
✗ Evitar !important

Evita usar !important. Si lo necesitas, es señal de que la estructura de tus estilos tiene problemas de especificidad.

/* ✗ Evitar */ .btn { color: red !important; } /* ✓ Mejor: aumenta especificidad */ .header .btn { color: red; }
✗ Evitar Selectores excesivos

No encadenes más de 3 niveles de selectores. Hace el CSS difícil de mantener y sobrescribir.

/* ✗ Demasiado específico */ nav ul li a span { color: pink; } /* ✓ Mejor: usar clase */ .nav-link { color: pink; }
✗ Evitar px fijos para fuentes

Usar px en fuentes impide que se escalen cuando el usuario ajusta el tamaño base del navegador.

/* ✗ No escala con el usuario */ body { font-size: 16px; } /* ✓ Respeta preferencias */ body { font-size: 1rem; } h1 { font-size: 2.5rem; }

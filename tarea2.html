Desarrollo Web

Los lenguajes de desarrollo web son los cimientos de todo lo que ves en internet. Se dividen en tecnologías de frontend (lo que el usuario ve) y backend (la lógica del servidor). Juntos hacen posibles desde blogs hasta plataformas globales.

HTML

HyperText Markup Language — La estructura de toda página web

Frontend · Markup

HTML es el lenguaje de marcado estándar para crear páginas web. Define la estructura y el contenido: párrafos, títulos, imágenes, enlaces y formularios. No es un lenguaje de programación, sino de marcado.

Creado por Tim Berners-Lee en 1991, su versión actual es HTML5, que añadió soporte nativo para audio, video, canvas y APIs semánticas.

Estructura web SEO Accesibilidad Formularios

Ejemplo de sintaxis

<!-- Estructura básica HTML5 --> <!DOCTYPE html> <html lang="es"> <head> <title>Mi página</title> </head> <body> <h1>Hola mundo</h1> <p>Bienvenido a la web.</p> <a href="#">Ver más</a> </body> </html>

CSS

Cascading Style Sheets — El estilo visual de la web

Frontend · Estilos

CSS controla la presentación visual de las páginas: colores, tipografías, espaciados, animaciones y layouts. Trabaja siempre en conjunto con HTML.

Con CSS3 llegaron Flexbox, Grid, variables CSS, transiciones y animaciones, que permiten crear interfaces modernas y responsivas sin JavaScript.

Diseño responsivo Animaciones Layouts Temas

Ejemplo de sintaxis

/* Variables y estilos globales */ :root { --color-rosa: #f7c5d5; --fuente: 'DM Sans', sans-serif; } .tarjeta { background: var(--color-rosa); border-radius: 12px; padding: 1.5rem; transition: transform 0.3s; } .tarjeta:hover { transform: translateY(-6px); }

JavaScript

El lenguaje de programación de la web — interactividad y lógica

Frontend + Backend

JavaScript es el único lenguaje de programación nativo de los navegadores. Permite añadir interactividad, validación, animaciones y comunicación con servidores en tiempo real.

Con Node.js también se usa en el backend. Frameworks como React, Vue y Angular lo convierten en el lenguaje más demandado del mundo.

Interactividad APIs React / Vue Node.js

Ejemplo de sintaxis

// Función con arrow function const saludar = (nombre) => { return `Hola, ${nombre}!`; }; // Fetch a una API async function obtenerDatos() { const res = await fetch('/api/datos'); const data = await res.json(); console.log(data); } saludar("Antonella"); // "Hola, Antonella!"

PHP

Hypertext Preprocessor — El lenguaje servidor de la web clásica

Backend · Servidor

PHP es un lenguaje de scripting del lado del servidor, diseñado específicamente para el desarrollo web. Potencia más del 75% de los sitios web del mundo, incluyendo WordPress.

Se integra directamente en HTML y se conecta fácilmente con bases de datos como MySQL. Es ideal para crear sistemas de login, formularios y gestores de contenido.

WordPress Bases de datos CMS Laravel

Ejemplo de sintaxis

<?php // Variables y condiciones $nombre = "Antonella"; $edad = 17; if ($edad >= 18) { echo "Mayor de edad"; } else { echo "Hola, $nombre!"; } // Consulta a base de datos $query = "SELECT * FROM usuarios"; ?>
Desarrollo de Escritorio

Los lenguajes de escritorio permiten crear aplicaciones que se instalan y ejecutan directamente en el sistema operativo, sin necesidad de internet. Son más potentes para tareas de alto rendimiento como videojuegos, software científico y herramientas de productividad.

Python

El lenguaje más versátil — IA, ciencia de datos y automatización

Escritorio · IA · Data

Python es el lenguaje de más rápido crecimiento en el mundo. Su sintaxis clara y legible lo hace ideal para principiantes y expertos por igual.

Es el lenguaje número uno en Inteligencia Artificial y Machine Learning, con bibliotecas como TensorFlow, PyTorch y scikit-learn. También se usa para automatización, scripts, scraping y desarrollo web (Django, Flask).

Inteligencia Artificial Automatización Data Science Django

Ejemplo de sintaxis

# Clase y herencia en Python class Estudiante: def __init__(self, nombre): self.nombre = nombre def presentarse(self): return f"Soy {self.nombre}" # Lista por comprensión notas = [8, 9, 10, 7] aprobados = [n for n in notas if n >= 8] alumna = Estudiante("Antonella") print(alumna.presentarse())

Java

Write once, run anywhere — El gigante empresarial

Escritorio · Android

Java es uno de los lenguajes más usados en el mundo empresarial. Su filosofía "escribe una vez, ejecuta en cualquier lugar" lo hace extremadamente portable gracias a la Java Virtual Machine (JVM).

Es el lenguaje base de Android, y se usa ampliamente en sistemas bancarios, aplicaciones empresariales y backend de grandes plataformas.

Android Empresarial Spring Boot Backend

Ejemplo de sintaxis

// Clase con método en Java public class Calculadora { public static int sumar(int a, int b) { return a + b; } public static void main(String[] args) { int resultado = sumar(5, 3); System.out.println( "Resultado: " + resultado ); // Resultado: 8 } }

C++

Alto rendimiento — Videojuegos, sistemas operativos y software crítico

Escritorio · Sistemas

C++ es uno de los lenguajes más potentes y eficientes. Al estar muy cercano al hardware, ofrece un rendimiento excepcional que lo hace indispensable en videojuegos, sistemas operativos y software en tiempo real.

Motores como Unreal Engine están escritos en C++. También se usa en software de Adobe (Photoshop), navegadores (Chrome) y sistemas embebidos.

Videojuegos Unreal Engine Sistemas Rendimiento

Ejemplo de sintaxis

// Programa básico en C++ #include <iostream> using namespace std; int main() { int contador = 0; for (int i = 0; i < 5; i++) { contador += i; } cout << "Suma: " << contador << endl; return 0; }
Diseño & Maquetación

El diseño digital no solo se hace con herramientas gráficas: existen lenguajes y formatos específicos para definir estilos, gráficos vectoriales, documentos y estructuras visuales de manera precisa y escalable.

SVG

Scalable Vector Graphics — Gráficos vectoriales para la web

Diseño · Vectorial

SVG es un formato de imagen basado en XML que describe gráficos vectoriales. A diferencia de PNG o JPG, los SVG se escalan sin perder calidad a cualquier tamaño.

Se usa para íconos, logotipos, ilustraciones e infografías en la web. Puede ser manipulado con CSS y JavaScript, y es editado con herramientas como Illustrator o Inkscape.

Íconos Logos Animaciones Infografías

Ejemplo de sintaxis

<!-- SVG: círculo animado --> <svg viewBox="0 0 100 100" xmlns="http://www.w3.org/2000/svg"> <circle cx="50" cy="50" r="40" fill="#f7c5d5" stroke="#b85c7e" stroke-width="3"/> <text x="50" y="55" text-anchor="middle" font-size="14"> Hola SVG </text> </svg>

Markdown

Lenguaje de marcado ligero — Documentación y contenido

Diseño · Documentación

Markdown es un lenguaje de marcado ligero que permite dar formato a texto plano con una sintaxis muy simple. Se convierte automáticamente a HTML.

Es usado en GitHub, Notion, Reddit, Discord y prácticamente toda herramienta de documentación moderna. Los archivos README de proyectos se escriben en Markdown.

GitHub Documentación Blogs Notion

Ejemplo de sintaxis

# Título principal (H1) ## Subtítulo (H2) Texto normal con **negrita**, *cursiva* y `código inline`. - Elemento de lista - Otro elemento [Enlace](https://www.ejemplo.com) ```python print("Hola Markdown") ``` > Cita en bloque

SASS / SCSS

CSS con superpoderes — Variables, mixins y anidamiento

Diseño · Preprocesador

SASS (con sintaxis SCSS) es un preprocesador CSS que extiende las capacidades del CSS estándar. Permite usar variables, anidamiento, mixins y funciones antes de que CSS los soportara nativamente.

Es ampliamente usado en proyectos profesionales con Bootstrap, Angular y grandes sistemas de diseño. El código SCSS se compila a CSS estándar para el navegador.

Bootstrap Variables Mixins Sistemas de diseño

Ejemplo de sintaxis

// Variables y mixins SCSS $primario: #f7c5d5; $radio: 12px; @mixin flex-center { display: flex; align-items: center; justify-content: center; } .boton { @include flex-center; background: $primario; border-radius: $radio; &:hover { background: darken($primario, 15%); } }
